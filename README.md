# 🎬 Crunchyroll-Inspired Authentication Flow

> A mobile-first authentication experience designed for a modern anime streaming platform. Built in Figma with a focus on clarity, speed, and frictionless user onboarding.

---

## 🚀 Product Vision

This project explores how a user can move from curiosity → signup → login in the shortest possible cognitive path.

The goal was simple:  
**reduce friction, increase visual delight, and maintain consistency across onboarding screens.**

---

## 🎯 Key Highlights

- ⚡ Minimal, distraction-free authentication flow  
- 📱 Mobile-first UI optimized for real-world usage  
- 🔄 Interactive prototype with seamless screen transitions  
- 🎨 Consistent design system (typography, spacing, colors)  
- 🧭 Clear onboarding journey: Splash → Login → Sign Up  

---

## 🧩 User Flow

1. **Splash Screen** → Brand introduction & first impression  
2. **Login Screen** → Returning user access  
3. **Sign Up Screen** → New user onboarding  

---

## 🛠️ Design Stack
